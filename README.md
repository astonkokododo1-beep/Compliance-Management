# Enterprise Compliance Management Platform

A comprehensive, AI-powered compliance management platform for enterprise organizations.

## Features

- **Policy Management & Intelligence Hub** - AI-powered policy drafting and gap analysis
- **Gift, Hospitality & Entertainment Management** - Digital declarations with AI risk scoring
- **Conflict of Interest Management** - Network mapping and continuous monitoring
- **Whistleblowing Investigation Manager** - Anonymous reporting with AI-assisted triage
- **Anti-Bribery & Anti-Corruption Detection** - Transaction surveillance and alerts
- **Insider Trading Surveillance** - Securities monitoring and pattern detection
- **Fraud Analytics Engine** - Multi-vector fraud detection

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Vercel Postgres + Prisma
- NextAuth.js
- Tailwind CSS + shadcn/ui
- Recharts
- Zustand
- Groq & Gemini AI APIs
